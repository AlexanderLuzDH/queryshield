Quick Start Guide

Django Users

Step 1: Add to your Django settings:
INSTALLED_APPS = [ 'queryshield_probe', # ... your other apps ]
Step 2: Run your tests:
python manage.py test
Step 3: Check the report:
cat queryshield_report.json

FastAPI Users

Step 1: Add to your app:
from queryshield_sqlalchemy import QueryShieldProbe from sqlalchemy import create_engine engine = create_engine("postgresql://...") probe = QueryShieldProbe(engine) # Use in tests... report = probe.finalize()

Production Monitoring

Step 1: Add middleware:
from queryshield_monitoring import QueryShieldMiddleware app.add_middleware( QueryShieldMiddleware, sample_rate=0.01, # 1% of queries slow_query_threshold_ms=500 )

What QueryShield Detects

‚ùå N+1 Queries: Loop that runs 1 query per item instead of batching
‚ùå Missing Indexes: Columns used in WHERE clauses without indexes
‚ùå Slow Queries: SELECT * from huge tables taking 500ms+
